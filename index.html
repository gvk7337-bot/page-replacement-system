The Three Algorithms at a Glance

// quick summary

FIFO — First In, First Out

Replaces the page that has been in memory the longest. Simple circular pointer. Can suffer from Belady's Anomaly.

LRU — Least Recently Used

Replaces the page not used for the longest time. Better real-world performance than FIFO. Immune to Belady's Anomaly.

Optimal — Belady's Algorithm

Replaces the page used farthest in the future. Theoretical minimum faults. Used as a benchmark only.
